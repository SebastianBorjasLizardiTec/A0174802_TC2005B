Border: Define el borde al rededor de un elemento
Border-left: Define el borde en el lado izquierdo.
Border-right: Define el borde en el lado derecho.
Border-top: Define el borde en el lado superior.
Border-bottom: Define el borde en la parte inferior del elemento.
Border-radius: Redondea las esquinas del borde del elemento.
Color: Esta propiedad establece el color del texto.
Font: Establece la fuente del texto.
Font-family: Define la familia de la fuente del texto del elemento.
Font-style: Define el estilo de la fuente.
Font-weight: Define el peso de la fuente.
width, height: Definen el ancho y la altura del elemento.
list-style: Define el estilo de los ítems en una lista.
margin: Define el espacio exterior alrededor de un elemento.
padding: Define el espacio interior dentro de un elemento.
overflow: Define qué hacer con el contenido que se desborda del área del elemento.
display: Define cómo se debe mostrar un elemento en el documento.
position: Define cómo se posiciona un elemento en el documento.
filter: Define efectos gráficos como el desenfoque o el brillo en un elemento.
